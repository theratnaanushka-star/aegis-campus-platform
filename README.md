# AEGIS — Unified Campus Digital Ecosystem

## Overview
AEGIS is a web platform that integrates campus services such as grievance management, academic resources, and opportunity tracking into a single role-based system.

## Problem
Campus tools are fragmented across multiple platforms, causing inefficiencies and lack of transparency.

## Solution
A unified dashboard with role-based access to manage campus workflows.

## Features
- Role-based authentication
- Grievance tracking system
- Academic resource hub
- Internship and opportunity portal

## Tech Stack
Frontend: React
Backend: Node.js + Express
Database: MongoDB
Authentication: JWT

Architecture defined, modules planned, development starting.
